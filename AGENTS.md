# Consignes locales — Concentric Magic Squares

Ce dépôt contient des scripts expérimentaux, pas une base éditoriale complète.

- Les générateurs et compteurs doivent être exécutés avec une borne explicite.
- Toute modification de définition doit mettre à jour le README et la note
  correspondante avant de régénérer des résultats.
- Les sorties réutilisables vont dans `results/`; les sorties locales ou
  volumineuses restent ignorées dans `generated/` ou `private/`.
- Ne pas présenter un comptage expérimental comme une classification générale
  sans préciser la borne et la méthode.
